Email Input

Demonstrating a simple email input field styled with Bootstrap. Notice that you can have a placeholder input there until the user types something.

<div class="emailform">
  <label for="emailInput" class="form-label">Enter your email:</label>
  <input type="email" class="form-control" id="emailInput" placeholder="user@example.com">
</div>

Multi-line Text Input

Bootstrap's textarea control helps you create resizable multi-line input fields for capturing longer text. Here, rows is set to 4.

<div class="mb-3">
  <label for="messageTextarea" class="form-label">Your message:</label>
  <textarea class="form-control" id="messageTextarea" rows="4" placeholder="Write your message here..."></textarea>
</div>
